# Design Spec: 시그널플레이 (SignalPlay)

> **소유자**: Designer 피카
> **최종 수정**: 2026-03-22
> **상태**: 초안 (Draft)

---

## 1. 디자인 방향

### 컨셉: "Dark Intelligence"

시그널플레이는 금융 데이터의 신뢰감과 게임의 몰입감을 동시에 전달해야 한다. 어두운 배경 위에 네온 액센트가 살아있는 시그널을 강조하고, 카드 기반 인터페이스가 정보를 소화하기 쉽게 분리한다.

**3가지 핵심 감성 키워드**:
1. **Sharp** — 데이터는 정확하고 날카롭게 표현
2. **Alive** — 애니메이션과 실시간성으로 살아있는 느낌
3. **Rewarding** — 예측 성공 시 명확한 피드백과 만족감

### 모드

- **기본**: 다크 모드 (Dark Mode First)
- **추후**: 라이트 모드 옵션 (Phase 2)

## 3. 타이포그래피

### 폰트 패밀리

| 역할 | 폰트 | 비고 |
|---|---|---|
| 본문, UI 텍스트 | **Pretendard** | 한국어 최적화, variable font |
| 숫자, 데이터, 코드 | **Space Grotesk** | Tabular nums, 금융 데이터용 |

### 스케일

| 토큰 | 크기 | 두께 | 용도 |
|---|---|---|---|
| `text-xs` | 11px | 400 | 배지, 타임스탬프 |
| `text-sm` | 13px | 400 | 보조 텍스트 |
| `text-base` | 15px | 400 | 본문 |
| `text-lg` | 17px | 500 | 카드 제목 |
| `text-xl` | 20px | 600 | 섹션 헤더 |
| `text-2xl` | 24px | 700 | 페이지 타이틀 |
| `text-3xl` | 30px | 800 | 점수, 숫자 강조 |

### 숫자 데이터 규칙

- 모든 가격/퍼센트 수치는 Space Grotesk 폰트 적용
- Tabular numbers (`font-variant-numeric: tabular-nums`) 필수 — 숫자 정렬 목적
- 양수: `--color-success` / 음수: `--color-danger` / 0: `--color-text-secondary`

## 4. 화면 목록

### 4.1 홈 — 시그널 피드 (`/`)

- 상단: 날짜 + 미해결 예측 수 배지
- 중단: AI 시그널 카드 수직 피드 (3~5개)
- 하단: 오늘의 AI 인사이트 미리보기 (수평 스크롤)
- FAB: "예측 제출" 퀵 액션 버튼

### 4.2 예측 게임 (`/game`)

- 시그널 카드 상세 (풀스크린 모달)
- 예측 슬라이더: 상승 / 횡보 / 하락
- 신뢰도 조절: 1~5단계 (콤보 배수)
- 제출 버튼 → 확인 애니메이션 (Lottie or Framer Motion)

### 4.3 리더보드 (`/leaderboard`)

- 주간 / 월간 탭
- 상위 10위 카드 (순위, 닉네임, 정확도%, 배지)
- 내 순위 고정 표시 (하단 sticky)

### 4.4 AI 인사이트 (`/insights`)

- 카테고리 필터 칩: 전체 / 매크로 / 종목 / 섹터 / 글로벌
- 인사이트 카드 그리드 (2열)
- 무한 스크롤

### 4.5 프로필 & 배지 (`/profile`)

- 사용자 아바타, 닉네임, 가입일
- 예측 통계: 총 참여 / 정확도 / 최고 연속 정답
- 배지 컬렉션 (획득 / 미획득)
- 설정 링크

## 5. 컴포넌트 스펙

### 5.1 SignalCard

```
┌─────────────────────────────────────┐
│ [섹터 칩] [임팩트 배지 HIGH]         │
│                                     │
│ 시그널 제목 (최대 2줄)               │
│                                     │
│ AI 요약 텍스트 (최대 3줄)           │
│                                     │
│ [예측 참여 →]          [마감 18시간] │
└─────────────────────────────────────┘
```

- **크기**: 전체 폭 × 가변 높이 (최소 120px)
- **배경**: `--color-bg-surface`
- **테두리**: 1px `--color-border`, 반경 12px
- **상태**: default / hover (elevated) / predicted (accent border) / expired (muted)
- **애니메이션**: 진입 시 fade-in + slide-up (Framer Motion, staggerChildren)

### 5.2 PredictionSlider

- 3단계 토글 버튼 (상승 / 횡보 / 하락)
- 선택 시 해당 색상으로 활성화 + 미세 진동(haptic) 애니메이션
- 신뢰도 단계: 별 아이콘 1~5개

### 5.3 LeaderboardRow

```
#1  🥇 [아바타] 닉네임        72.4%  [배지]
```

- 1~3위: 금/은/동 아이콘 + 배경 그라데이션
- 내 행: `--color-primary` 배경 tint

### 5.4 InsightCard

- 2열 그리드, 가로 폭 (100% - 32px padding) / 2
- 상단: 카테고리 칩 + 저장 아이콘
- 본문: 제목 (2줄) + 요약 (3줄)
- 하단: 소스 출처 + 시간

### 5.5 BadgeChip

- 소형 (20px × 20px): 리더보드 행에 사용
- 중형 (36px × 36px): 프로필 배지 컬렉션
- 대형 (64px × 64px): 배지 획득 팝업

## 6. 접근성

### 기준

- **WCAG 2.1 AA** 준수 목표
- 색상 대비: 텍스트 최소 **4.5:1** (소형), 대형 텍스트 3:1
- 시그널 색상(상승/하락)은 색상만으로 구분하지 않음 — 아이콘(↑↓→) 병행 표시

### 구현 체크리스트

- [ ] 모든 인터랙티브 요소 `tabIndex` 설정
- [ ] 아이콘 버튼에 `aria-label` 필수
- [ ] 이미지에 `alt` 텍스트
- [ ] 모달 열릴 때 포커스 트랩
- [ ] 시스템 `prefers-reduced-motion` 감지 → 애니메이션 최소화

## 8. 디자인 QA 체크리스트

### 출시 전 필수 확인

**레이아웃**
- [ ] 375px (iPhone SE), 390px (iPhone 15), 412px (Galaxy S24) 세 해상도 확인
- [ ] Safe area (노치, 홈 인디케이터) 침범 없음
- [ ] 스크롤 시 fixed 요소와 콘텐츠 겹침 없음

**색상 & 대비**
- [ ] 모든 텍스트 WCAG AA 색상 대비 통과
- [ ] 상승/하락 색상을 색맹 필터(Deuteranopia)로 확인

**타이포그래피**
- [ ] 숫자 데이터에 Space Grotesk 적용 확인
- [ ] 긴 텍스트 말줄임(`text-overflow: ellipsis`) 정상 작동

**인터랙션**
- [ ] 탭 영역 최소 44×44px
- [ ] 로딩 상태 (Skeleton) 모든 카드에 구현
- [ ] 에러 상태 UI 구현 (빈 피드, 네트워크 오류)

**다크 모드**
- [ ] 시스템 다크/라이트 전환 시 깜빡임 없음
- [ ] 이미지 위 텍스트 가독성 확인
