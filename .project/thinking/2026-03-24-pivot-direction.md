# 피벗 방향 검토 — 2026-03-24

## 참고 프로젝트 분석

### Market Radar (market-dashboard-v2)
- **뭐하는 서비스:** 국내주식 52 + 미국주식 60 + 코인 250 종목 실시간 통합 대시보드
- **핵심 가치:** 흩어진 투자 정보를 한 곳에 모아 빠른 의사결정
- **데이터:** 한국투자증권, Yahoo Finance, Upbit, CoinPaprika 등 10개+ 소스
- **AI:** Gemini로 뉴스 자동 요약 + 종목 연결
- **특징:** 워치리스트, 급등/급락 랭킹, 섹터 로테이션, 캔들 차트
- **기술:** React 19 + Vite + Tailwind + React Query

### 침팬지픽 (chimp-pick)
- **뭐하는 서비스:** 실제 시세 기반 UP/DOWN 바이너리 예측 게임
- **핵심 가치:** 실투자 부담 없이 예측 재미 + B급 밈 감성
- **게임 루프:** 종목 선택 → 타임프레임(1분/5분/1시간/1일) → UP/DOWN → 바나나코인 베팅 → 결과
- **경제:** 바나나코인 (가상 재화), 출석/연승 보너스, 인앱 구매
- **랭킹:** 침팬지 진화 단계 (원숭이→침팬지→고릴라→킹콩→전설의침팬지)
- **기술:** 실시간 WebSocket 시세, PWA, 다크 모드

## 두 프로젝트가 말해주는 것

대표님은 이미 두 가지를 별도로 만들고 있다:
1. **Market Radar** = "진지한" 투자 정보 도구 (대시보드, 실시간 시세, 차트)
2. **침팬지픽** = "재밌는" 투자 예측 게임 (B급, 바이너리 베팅, 밈 감성)

**시그널플레이는 이 두 개 사이 어딘가에 있다.** 진지하지도 않고 재밌지도 않은 애매한 포지션.

## 시그널플레이만의 포지션은?

### Market Radar와의 차이
- Market Radar는 "모든 종목 실시간 대시보드" → 무거움
- 시그널플레이는 앱인토스 안에서 "30초에 끝나는 가벼운 경험" → 가벼움
- Market Radar의 데이터 인프라(시세 API, 종목 DB)를 시그널플레이가 활용할 수 있음

### 침팬지픽과의 차이
- 침팬지픽은 "실시간 시세 기반 단타 예측" → 도박 느낌, 실시간 긴장
- 시그널플레이는 "일 1회 경제 동향 예측" → 습관 형성, 학습 느낌
- 침팬지픽은 PWA 독립 서비스, 시그널플레이는 앱인토스 미니앱

### 시그널플레이만의 기회
앱인토스라는 플랫폼 특성:
- **토스 유저 3000만 즉시 접근** (설치 불필요)
- **토스 로그인 1탭** (가입 허들 제로)
- **토스 리더보드/광고 SDK** 무료 제공
- **하지만** 토스 앱 안이라 무거운 기능은 부적합

## 피벗 방향 (프로젝트 위에서)

### 핵심 재정의

기존: "AI가 만든 경제 시그널로 예측 게임"
→ 문제: 왜 예측해야 하는지 이유가 없음

**피벗안: "오늘 시장 30초 브리핑 + 내 의견 남기기"**

Market Radar의 데이터 + 침팬지픽의 게임성 + 앱인토스의 가벼움

구체적으로:
1. **오늘의 시장 한줄 브리핑** — Market Radar 데이터 기반, AI가 "오늘 코스피 +1.2%, 반도체가 끌었다" 수준으로 정리
2. **오늘의 토론 주제** — "삼성전자 12만원 갈까?" 같은 구체적 종목 기반 질문
3. **내 의견 투표** — 호재/악재가 아니라 "간다/안간다/모르겠다" + 한줄 의견
4. **군중 심리 실시간** — "73%가 간다에 투표" → 이게 보러 다시 옴
5. **다음날 결과** — 실제 주가와 비교 → "군중이 맞았다/틀렸다"

### 왜 이게 다른가

- **구체적이다** — "연준 금리" 같은 뜬구름이 아니라 "삼성전자 12만원" 같은 구체적 질문
- **내 관심사다** — 사람들이 실제로 들고 있거나 관심 있는 종목
- **의견을 남긴다** — 3지선다 게임이 아니라 내 생각을 표현하는 행위
- **다른 사람 의견이 궁금하다** — 군중 심리가 실시간으로 바뀌니까 보러 옴
- **결과가 궁금하다** — 내가 투표한 종목의 실제 결과를 확인하러 옴

### Market Radar 데이터 활용 가능성

Market Radar가 이미 구축한 것:
- 한국투자증권 API (국내주식 실시간)
- Yahoo Finance API (미국주식)
- Upbit/CoinPaprika API (코인)
- 뉴스 RSS + Gemini 요약

시그널플레이가 Market Radar의 API를 호출하거나, 같은 데이터 소스를 공유하면:
- "오늘 코스피 급등 TOP 5" → 이 중 하나를 토론 주제로
- "삼성전자 오늘 +3.2%" → 구체적 데이터 기반 브리핑
- "이번 주 가장 핫한 섹터: 반도체" → 섹터 기반 토론
