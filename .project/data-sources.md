> **소유자**: BE 볼트
> **최종 수정**: 2026-03-22
> **상태**: 초안 (Draft) — 앱인토스 피벗 반영

---

## 1. 데이터 전략 개요

시그널플레이는 **Gemini API가 뉴스를 분석**해 시그널 카드를 생성한다. 외부 API 의존성을 최소화하고, 무료로 운용 가능한 소스만 사용한다.

- **시세 데이터 사용 안 함**: 실시간 주가/지수 데이터는 거래소 라이선스 필요 + 비용 발생. MVP에서 완전 제외.
- **뉴스 RSS 피드 우선**: API 키 없이 접근 가능한 RSS 피드를 1차 소스로 사용.
- **Gemini가 판단**: 수집된 뉴스 헤드라인을 Gemini에게 전달하면 시그널 생성 + 결과 평가 모두 처리.

### 2.3 수집 방식

Vercel Cron Job (`/api/cron/generate`, 매일 08:00 KST) 실행 시:

```
1. 위 RSS 피드 순회 — 각 피드에서 최근 6시간 기사 헤드라인 + URL 수집
2. 중복 URL 제거
3. 헤드라인 목록 (최대 30개) → Gemini API 프롬프트에 전달
4. Gemini가 경제지표 / 기업동향 / 글로벌이슈 카테고리로 분류 + 시그널 3개 선별 및 요약
```

## 3. 시세 데이터 — 사용 안 함

### 이유

- 실시간 주가/지수 데이터는 KRX, NYSE 등 거래소의 별도 라이선스 필요.
- 일간 종가(EOD) 데이터도 상업 서비스에서 무료로 재배포하려면 API 이용 약관 검토 필요.
- **MVP 전략**: 시그널 카드는 "이 뉴스가 호재냐 악재냐"를 예측하는 것이지, 주가 등락을 예측하는 것이 아님. 시세 데이터 불필요.

### 결과 판단 방식

- `actual_result` (호재/악재/영향없음)는 **다음날 Gemini가 후속 뉴스를 보고 평가**한다.
- 예: "어제 '연준 금리 동결 시사' 뉴스에 대해 오늘 시장 반응 관련 뉴스를 분석해 호재/악재/영향없음을 판단하라"

```
Gemini 결과 평가 프롬프트:
"다음은 어제의 시그널 뉴스입니다: {어제_뉴스_요약}
오늘 이와 관련된 후속 뉴스는 다음과 같습니다: {오늘_관련_뉴스_헤드라인}
이 시그널이 결국 경제/시장에 '호재', '악재', '영향없음' 중 어느 것이었는지 판단하고 이유를 30자 이내로 설명하세요."
```

## 4. 논문/리서치 — 사용 안 함 (MVP)

MVP에서는 학술 논문(arXiv 등) 데이터를 사용하지 않는다. 뉴스 기반 시그널로 충분하며, 논문 기반 인사이트는 서비스 방향성과 맞지 않는다(일반인 대상 경제 동향 요약). Phase 3 고도화 시 재검토.

## 5. 법적 고려

### RSS 피드 이용

- RSS 피드는 뉴스사가 공개 배포용으로 제공하는 것이나, 뉴스 본문 전체를 DB에 저장하고 재배포하는 것은 저작권 침해 가능성.
- **MVP 전략**: 헤드라인 + URL만 수집. Gemini에게 헤드라인만 전달해 요약 생성. 원문 본문은 저장하지 않고 `source_url`(링크)만 시그널 카드에 포함.
- 시그널 카드의 요약 텍스트는 Gemini가 생성한 새로운 텍스트 — 원문 재배포 아님.

### 앱인토스 콘텐츠 규정

- 암호화폐 관련 뉴스는 RSS 수집 단계에서 Gemini 프롬프트로 필터링.
- 특정 종목/상품 추천 표현은 Gemini 프롬프트 제약 조건으로 제거.
- 모든 시그널 카드에 "AI 생성 콘텐츠" 레이블 + `source_url` 원문 링크 표시.

## 6. 데이터 보존 정책

| 테이블 | 보존 기간 | 삭제 방법 |

## 7. 품질 확보

### 신뢰 출처 우선

Gemini 프롬프트에 신뢰 출처 화이트리스트를 제공해 해당 출처의 뉴스를 우선 선택하도록 유도:

```typescript
const TRUSTED_SOURCES = [
  '한국경제', '매일경제', '연합뉴스', '이데일리',
  'Reuters', 'BBC', 'Financial Times',
  '한국은행', '금융위원회', 'Federal Reserve', 'ECB',
]
```

### 암호화폐 필터링

Gemini 프롬프트에 명시적 제외 지시:
```
다음 키워드가 포함된 뉴스는 시그널에서 제외하세요:
비트코인, 이더리움, 암호화폐, 가상자산, NFT, 코인, 블록체인 투자
```

### Hallucination 방지

- 시그널 요약은 제공된 뉴스 헤드라인에 기반해서만 작성. 임의 정보 추가 금지.
- Gemini 프롬프트: "제공된 뉴스 헤드라인에 없는 정보는 추가하지 마세요."
- 시그널 카드에 `source_url` 원문 링크 표시 → 사용자가 원문 확인 가능.
