- [x] 팀 구성 및 에이전트 파일 작성

- [ ] Supabase 프로젝트 생성 및 스키마 설계
- [ ] CI/CD 파이프라인 구축 (GitHub Actions + Vercel)

## Phase 1: MVP 개발 (Week 1-4)
**목표:** 핵심 게임 루프 검증 — "시그널 확인 → 예측 → 결과 확인"

### 핵심 기능
- [ ] 소셜 로그인 (카카오, 구글) + 익명 모드
- [ ] Daily Signal: AI가 생성한 오늘의 투자 시그널 3~5개
- [ ] 예측 게임: 시그널에 대해 상승/하락 예측
- [ ] 결과 확인: 예측 정확도 피드백 + 포인트 지급
- [ ] 리더보드: 주간/월간 예측 왕 랭킹
- [ ] AI 인사이트 피드: 짧은 투자 분석 카드

### 기술 마일스톤
- [ ] Supabase 스키마 + RLS 정책 완료
- [ ] OpenAI Edge Function으로 시그널 생성 파이프라인
- [ ] 핵심 UI 컴포넌트 (SignalCard, PredictionSlider, LeaderboardRow)
- [ ] E2E 테스트 (회원가입 → 예측 → 결과 확인 플로우)

## Phase 2: 제품 완성 (Week 5-8)
**목표:** 리텐션 강화 + 디자인 고도화 + 안정성 확보

- [ ] 뱃지/업적 시스템 (예측 연속 정답, 분야별 전문가 등)
- [ ] 푸시 알림 (오늘의 시그널, 예측 결과)
- [ ] Breaking News Catch 게임 (속보 빠르게 캐치하면 보너스)
- [ ] AI 인사이트 고도화 (논문 기반 분석, 과거 데이터 패턴)
- [ ] 다크/라이트 모드 완성
- [ ] 성능 최적화 (Lighthouse 90+)
- [ ] 보안 감사 및 취약점 패치

## Phase 3: 성장 & 수익화 (Week 9-12)
**목표:** 사용자 획득 + 수익 모델 검증

- [ ] 광고 통합 (배너, 네이티브 광고)
- [ ] 프리미엄 구독 (Pro Signal: 고급 AI 분석, 알림 커스텀)
- [ ] 소셜 기능 (예측 공유, 친구 초대, 그룹 리더보드)
- [ ] 토스 미니앱 정식 배포
- [ ] 사용자 피드백 수집 및 반영
- [ ] A/B 테스트 인프라 구축

## Phase 4: 장기 전략 (Week 13+)
**목표:** 플랫폼 확장 + 파트너십

- [ ] 글로벌 시장 데이터 지원 (미국 주식, 글로벌 코인)
- [ ] 파트너십 (증권사, 금융 교육 플랫폼)
- [ ] 커뮤니티 기능 (투자 토론, 시그널 공유)
- [ ] AI 모델 고도화 (자체 미세 조정 모델)
- [ ] 실시간 데이터 확대 (프리미엄)
