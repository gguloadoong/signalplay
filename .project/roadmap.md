# 시그널플레이 (SignalPlay) — Product Roadmap

> 앱인토스 피벗 기준 로드맵. 3개 화면 MVP를 1주 안에 출시하는 것을 목표로 한다.

**목표**: 개발 환경 구축 + 앱인토스 접근 확보

- [x] 프로젝트 디렉터리 및 GitHub 레포 생성
- [x] 팀 구성 및 에이전트 파일 작성
- [x] 프로젝트 문서 초안 작성 (PRD, 디자인 스펙, 기술 스펙 등) — 앱인토스 피벗 반영
- [ ] 앱인토스 개발자 등록 및 Bedrock SDK 접근 키 발급 (제이크 담당)

- [ ] Supabase 프로젝트 생성 + 3테이블 스키마 적용

## Phase 1: MVP (Week 1)

**목표**: 핵심 게임 루프 출시 — "시그널 확인 → 예측 → 다음날 결과 확인"

### 핵심 기능 (3개 화면)

- [ ] **토스 로그인**: Bedrock SDK 연동. `toss_user_id` 획득.
- [ ] **홈 화면**: 오늘의 시그널 카드 3개 + 호재/악재/영향없음 예측 버튼
- [ ] **결과 화면**: 어제 예측 결과 확인 + 점수/스트릭 표시 + 리워드 광고 CTA
- [ ] **리더보드 화면**: 토스 리더보드 SDK 연동 주간 랭킹
- [ ] **면책 문구**: 홈/결과 화면 하단 고정 노출

### 백엔드 기능

- [ ] Supabase 3테이블 + RLS 정책 완성
- [ ] Vercel Serverless Functions: `/api/signals`, `/api/predictions`, `/api/results`, `/api/stats`
- [ ] Vercel Cron Job: 매일 08:00 KST Gemini API로 시그널 3개 자동 생성
- [ ] Vercel KV 캐싱: 시그널 캐시 + 중복 제출 방지
- [ ] 점수 계산 로직 (정답 +10, 스트릭 보너스 +5×N일)
- [ ] AdMob 리워드 광고 연동 (Bedrock SDK, 결과 화면)

### 기술 마일스톤

- [ ] Bedrock SDK 로컬 모킹 완성 (WebView 외 환경 대응)
- [ ] MSW 핸들러 작성 (개발/CI 환경 목데이터)
- [ ] 3개 핵심 E2E 테스트 (Playwright, Bedrock 모킹)
- [ ] 앱인토스 WebView 호환성 검증 (Safe Area, 탭 영역, SDK 동작)

### 앱인토스 심사 제출

- [ ] 앱인토스 규정 준수 체크리스트 전항목 통과
- [ ] 면책 문구 노출 위치 및 텍스트 확인
- [ ] Gemini 프롬프트 암호화폐/투자 권유 제외 동작 검증

## Phase 2: 고도화 (Week 2~3)

**목표**: 리텐션 강화 + 공유 기능 + 광고 최적화

- [ ] **스트릭 강화**: 스트릭 시각화 (불꽃 이모지, D+N 배지), 스트릭 위기 알림
- [ ] **결과 공유**: 결과 카드 이미지 생성 → 토스 공유 시트 연동
- [ ] **전면 광고**: 리더보드 진입 시 AdMob 전면 광고 (1회/일, Bedrock SDK)
- [ ] **푸시 알림**: 토스 알림 SDK 연동 — 매일 오전 "오늘의 시그널 도착!" 알림
- [ ] **과거 시그널 아카이브**: 지난 7일 시그널 + 내 예측 기록 조회
- [ ] **다크 모드**: `prefers-color-scheme` 감지 + TDS 다크 토큰 적용
- [ ] **성능 최적화**: Lighthouse 90+, JS 번들 200KB 이하

---

## Phase 3: 수익화 (Week 4~6)

**목표**: 광고 수익 최적화 + 프리미엄 기능 검토

- [ ] **광고 최적화**: A/B 테스트 — 리워드 광고 위치, CTA 문구 변경으로 시청률 극대화
- [ ] **배지/업적 시스템**: 7일 스트릭 배지, 분야별 예측왕 배지 등 참여 유인 강화
- [ ] **프리미엄 구독 검토**: 광고 제거 + 시그널 5개 (월 1,900원, 토스페이 결제)
- [ ] **시그널 품질 고도화**: Gemini 프롬프트 개선, 뉴스 소스 다양화, 실제 결과 정확도 측정
- [ ] **A/B 테스트 인프라**: Vercel Edge Config 기반 플래그 시스템

---

## Phase 4: 장기 전략 (Week 7+)

**목표**: 앱인토스 인기 미니앱 안착

- [ ] 친구 초대 기능 (토스 연락처 연동, 그룹 리더보드)
- [ ] 앱인토스 메인 노출 협의 (토스 에디터픽)
- [ ] 시그널 카테고리 확장 (부동산, 원자재, 글로벌 지표)
- [ ] 기업/파트너십 제휴 (금융 교육 콘텐츠, 경제 미디어 협업)
- [ ] 사용자 제안 시그널 (커뮤니티 기능 검토)
