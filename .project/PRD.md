> **소유자**: PM 제이크
> **최종 수정**: 2026-03-22
> **상태**: 초안 (Draft)

## 1. 배경 & 문제 정의

### 배경

개인 투자자 수가 급증하면서 투자 정보에 대한 수요도 폭발적으로 늘었다. 그러나 대부분의 기존 서비스는 전문 투자자 중심으로 설계되어 있어, 투자 입문자나 중급자가 일상적으로 활용하기 어렵다.

### 핵심 문제

1. **투자 정보의 분산**: 뉴스, 논문, 시장 데이터, SNS가 분산되어 있어 개인 투자자가 의사결정에 필요한 정보를 직접 수집·정리하는 데 과도한 시간이 소요된다.
2. **정보 소비의 지루함**: 텍스트 중심의 금융 정보는 읽기 어렵고 흥미를 유발하지 못한다. 결국 지속적인 팔로업을 포기하게 된다.
3. **꾸준한 습관 형성의 어려움**: 투자는 장기 게임이지만, 매일 정보를 확인하고 시장 감각을 유지하는 루틴을 만들기 어렵다.
4. **신호와 소음의 구분 불가**: 방대한 정보 중 실제 투자 시그널로서 의미있는 것이 무엇인지 초보 투자자가 판단하기 매우 어렵다.

### 솔루션 가설

AI가 논문·뉴스·시장 데이터를 분석해 **투자 시그널을 카드 형태로 요약**하고, 유저가 해당 시그널에 대해 **예측 게임으로 참여**하게 하면:
- 정보 소비의 진입 장벽이 낮아지고
- 게임 메커니즘이 꾸준한 재방문을 유도하며
- 예측 결과를 통해 자연스럽게 시장 감각을 키울 수 있다.

## 2. 타겟 사용자

### Primary: 2030대 투자 입문~중급자

- **연령**: 25~39세
- **특성**: MZ세대, 모바일 네이티브, 게임·챌린지에 친숙, 토스 앱 사용자
- **투자 경험**: 주식 계좌 보유, ETF·개별 종목 소량 투자 경험 있음
- **페인포인트**: 바빠서 매일 시장을 확인하기 어렵고, 어떤 정보를 믿어야 할지 모르겠다
- **사용 시나리오**: 출퇴근 중 3~5분, 점심시간에 오늘의 시그널 확인 + 예측 참여

### Secondary: 40대 투자자

- **연령**: 40~49세
- **특성**: 어느 정도 투자 경험, 정보의 질을 중요시
- **사용 시나리오**: AI 인사이트 카드로 빠르게 시장 요약 소비, 심층 분석 리포트 열람

### Anti-persona: 전문 트레이더

- **이유**: 이미 Bloomberg, 퀀트 툴 등 전문 도구 사용. 당사 서비스의 데이터 깊이가 그들의 필요를 충족하지 못함. 이들을 위한 설계는 핵심 타겟 경험을 복잡하게 만들 위험이 있음.

## 3. 경쟁 분석

| 서비스 | 강점 | 약점 | 우리와의 차별점 |
|---|---|---|---|
| **토스증권** | 간편 UI, 대규모 사용자 기반 | 정보 깊이 부족, 게임성 없음 | 게임 메커니즘 + AI 시그널 |
| **네이버 금융** | 방대한 정보량, 높은 신뢰도 | 복잡한 UI, 모바일 UX 나쁨 | 모바일 퍼스트, 큐레이션 |
| **알파스퀘어** | AI 분석, 정량 데이터 강점 | 학습 곡선 높음, 전문가 중심 | 게이미피케이션, 낮은 진입 장벽 |
| **증권플러스** | 종목 토론, 커뮤니티 | 정보 신뢰도 이슈, 소음 많음 | AI 검증된 시그널, 큐레이션 |
| **StockTwits** | 글로벌, 소셜 | 한국 시장 부적합, 영어 | 한국 시장 특화, 한국어 AI |

**결론**: 게임성(예측·리더보드) + AI 큐레이션 + 모바일 UX를 조합한 서비스는 현재 국내에 없다.

## 4. 핵심 기능 명세

### 4.1 Daily Signal Game (핵심 기능)

- **설명**: 매일 오전 9시, AI가 당일 주목할 투자 시그널 3~5개를 생성. 유저는 각 시그널에 대해 "상승/하락/횡보" 예측을 제출.
- **결과 확인**: 다음 날 오전, 실제 시장 데이터 기반으로 예측 결과 및 점수 공개.
- **점수 체계**: 정확도 + 신뢰도(배팅 강도) 기반 점수 산정. 연속 정답 시 콤보 보너스.

### 4.2 AI Insight Cards

- **설명**: 뉴스·논문·시장 데이터를 AI가 분석해 150자 내외의 인사이트 카드로 요약.
- **카드 타입**: 매크로 이슈, 개별 종목, 섹터 트렌드, 글로벌 이벤트
- **상세 보기**: 카드 탭 시 원문 링크, AI 분석 근거, 관련 과거 시그널 제공.

### 4.3 Prediction Leaderboard

- **설명**: 주간·월간 예측 정확도 기반 랭킹. 닉네임·아바타로 익명성 보장.
- **배지 시스템**: 연속 정답, 특정 섹터 전문가, 얼리버드 등 업적 배지.
- **소셜 요소**: 상위 10명의 예측 포트폴리오 공개(커뮤니티 학습 효과).

### 4.4 Breaking News Catch

- **설명**: 시장에 영향을 줄 수 있는 Breaking News 실시간 알림. 푸시 알림 + 인앱 배너.
- **AI 필터링**: 소음성 뉴스 제거, 임팩트 점수(High/Medium/Low) 부여.

### 4.5 Cross-asset Dashboard

- **설명**: 주식, ETF, 채권, 원자재, 암호화폐의 간략한 동향을 한 화면에서 확인.
- **MVP 축소판**: 주요 지수(코스피, S&P500, 나스닥) + 환율 + 금/유가만 표시.

## 5. MVP 범위

### IN (MVP에 포함)

- Daily Signal Game (예측 제출 + 결과 확인)
- AI Insight Cards (일 5~10장)
- Prediction Leaderboard (주간 랭킹)
- 소셜 로그인 (카카오, 구글) + 익명 모드
- 기본 프로필 및 배지 2~3종

### OUT (MVP 이후)

- 실시간 시세 스트리밍 (비용·규제 이슈)
- 실제 매매 연동 (금융 라이선스 필요)
- 소셜 트레이딩 (팔로우, 포트폴리오 공유)
- 포트폴리오 트래커
- Breaking News 푸시 알림 (Phase 2)
- 프리미엄 구독 (Phase 3)

## 8. 리스크 & 가정

### 규제 리스크

- **투자 자문 vs 정보 제공**: AI 시그널이 투자 자문으로 해석될 경우 금융투자업 라이선스 필요. → 모든 콘텐츠에 "투자 참고용 정보이며 투자 책임은 본인에게 있습니다" 면책 문구 필수.
- **게임화와 도박법**: 예측 게임에 실물 보상(현금, 포인트 환전) 연결 시 도박법 위반 가능성. → MVP에서는 리더보드 순위와 배지만 보상으로 제공.

### 데이터 리스크

- **외부 API 의존도**: 뉴스 API, 시장 데이터 API의 서비스 중단 시 콘텐츠 생성 불가. → 최소 2개 이상의 데이터 소스 이중화.
- **AI 환각(Hallucination)**: OpenAI API가 부정확한 정보를 생성할 수 있음. → 인사이트 카드에 "AI 생성 콘텐츠" 명시, 원문 소스 링크 의무화.

### 기술 가정

- Supabase Edge Functions로 OpenAI API 비용을 서버에서 통제 가능하다.
- 토스 미니앱 환경에서 Next.js 15 기반 웹앱이 네이티브 수준의 성능을 제공할 수 있다.

## 9. 비즈니스 모델

### 광고 (Phase 1~2)

- **배너 광고**: 인사이트 카드 피드 사이 노출. 금융·핀테크 광고주 우선.
- **네이티브 광고**: AI 인사이트 카드 형태로 자연스럽게 삽입. 스폰서드 명시 필수.

### 프리미엄 구독 — Pro Signal (Phase 3)

- **가격**: 월 4,900원 (연간 39,900원)
- **혜택**:
  - 일 시그널 10개 이상 (무료: 5개)
  - 심층 AI 분석 리포트
  - 예측 히스토리 무제한 조회
  - Breaking News 우선 알림
  - 광고 없음

### 제휴 (Phase 4)

- 증권사 계좌 개설 CPA (Cost Per Acquisition) 제휴
- ETF/펀드 상품 소개 수수료
