## 1. 문제 정의

### 투자자의 진짜 가려운 부분

투자자는 정보가 부족한 게 아니다. **확신이 없다.**

- "이 뉴스가 내 종목에 호재야 악재야?" → 해석 능력 부족
- "나만 이렇게 생각하는 건가?" → 고립된 판단의 불안
- "뭔가 놓치고 있는 거 아냐?" → 정보 FOMO
- "다른 사람들은 어떻게 보고 있지?" → 집단 해석에 대한 갈증

토스증권은 **데이터(시세, 차트, 뉴스)**를 준다. 하지만 **"다른 투자자들의 해석"**은 안 준다.
네이버 금융 댓글은 노이즈 투성이. 종토방은 편향적. 유튜버는 1인 의견.

### 솔루션

> 시장에 무슨 일이 생기면 **가장 먼저 알려주고**, 방구석 전문가들이 **각자 다른 이론/알고리즘으로 예측**하고, 토스 유저들이 **집단 투표**로 해석하고, 그 판단이 **실제로 맞았는지 추적**해주는 곳.

## 3. 방구석 전문가 캐릭터 (ADR-008)

단순 AI 요약이 아니다. 각 캐릭터가 **논문, 이론, 알고리즘에 기반한 고유 방법론**으로 종목/지수/거시경제를 예측한다. 2030 투자자들이 자신 혹은 주변인을 투영하는 B급 감성 페르소나.

| 캐릭터 | 방법론 | 성격 | 예측 스타일 |
|--------|--------|------|-----------|
| 💼 **엑셀형** | PER/PBR/EV/EBITDA 밸류에이션 + N분기 실적 트렌드 | 스프레드시트에 삶을 건 직장인 | "PBR 0.7배 저평가, 3분기 연속 영업이익 개선 추세" |
| 📚 **도서관형** | 학술 논문 (PEAD, Fama-French, Momentum Factor) | 논문 3000편 읽고 주식 시작 | "Bernard & Thomas PEAD 모델 기준, 실적 서프라이즈 후 60일 양의 드리프트 예상" |
| 📺 **뉴스형** | 뉴스 센티멘트 점수화 + 시장 컨센서스 괴리 | 24시간 경제뉴스만 보는 사람 | "긍정 뉴스 비율 83%, 외국인 순매수 전환. 시장 컨센서스 상회 가능" |
| 📐 **차트형** | Golden Cross/Head & Shoulders + RSI/MACD + 지지·저항 레벨 | 차트에서 운명을 보는 사람 | "20일선 골든크로스 임박. 볼린저밴드 상단 55,000 돌파 시 추가 상승" |
| 🎲 **운형** | 완전 랜덤 (날씨·기분·꿈) | 그냥 느낌으로 찍는 사람 | "🎲 오늘 아침 날씨가 맑아서 호재인 것 같아요 (근거: 없음)" |

### 캐릭터 설계 원칙

- 실제 투자 분석 방법론에 기반. 근거 없는 의견 금지 (운형 제외).
- 같은 이슈에 서로 다른 결론이 나와야 함. 전원 호재면 재미없음.
- 예측에 반드시 근거 1줄 이상. "호재" 한 마디 금지 (운형 제외).
- 운형은 베이스라인. 다른 캐릭터가 운형보다 못하면 바이럴 콘텐츠.

### 적중률 리더보드

```
이번 달: 💼엑셀형 72% > 📚도서관형 65% > 📺뉴스형 61% > 🎲운형 58% > 📐차트형 54%
"운형이 차트형을 이겼다 ㅋㅋ"
```

## 5. 핵심 기능 (MVP)

### 5.1 오늘의 질문 + 투표
- 매일 오전 9시: 의견 갈릴 만한 이슈 기반 질문 1~2개 (Gemini 생성)
- 구체적 종목/이벤트 기반 ("삼성전자 실적 — 호재?", "코스피 2,800 간다?")
- 투표: 호재 / 악재 / 글쎄 (5초)
- 투표 후 군중 비율 공개 ("67%가 호재"). **투표해야 비율이 보인다.**

### 5.2 방구석 전문가 예측
- 각 질문에 5캐릭터 서로 다른 예측 + 방법론 기반 근거
- Gemini 캐릭터별 페르소나 프롬프트로 생성

### 5.3 결과 + 적중률 트랙레코드
- 다음날: 실제 시장 vs 군중 vs 캐릭터 비교
- 🎯 적중 / ❌ 빗나감 + 해설
- 캐릭터 리더보드 + 내 적중률

### 5.4 공유 카드 (바이럴)
- 투표 카드: 질문 + 군중 비율 + "너는 어떻게 봐?"
- 결과 카드: 적중/빗나감 + 캐릭터 리포트
- 카카오톡/인스타 공유

## 6. MVP 범위

**IN**: 질문+투표+군중, 5캐릭터 예측, 결과+적중률, 공유 카드, 토스 로그인, AdMob, 면책문구

**OUT (Phase 2+)**: 플래시 투표(속보), 푸시 알림, 관심 종목 개인화, 캐릭터 구독, 한줄 의견, 토스 포인트

## 7. 사용자 여정

```
[첫 진입]
  토스 → 시그널플레이 → 토스 로그인(1탭)
  → 오늘의 질문 ("삼성전자 실적 — 호재?")
  → 방구석 전문가 의견 (엑셀형: 호재, 차트형: 주의, 운형: 악재)
  → 투표 (5초) → 군중 67%가 호재
  → 공유 ("너는 어떻게 봐?" → 카톡)

[다음날]
  → 결과 (군중 맞았다 🎯, 침팬지 또 맞혔다 ㅋ)
  → 오늘의 새 질문 → 반복
```
