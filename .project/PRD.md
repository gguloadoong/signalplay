## 1. 문제 정의

### 투자자의 진짜 가려운 부분

투자자는 정보가 부족한 게 아니다. **확신이 없다.**

- "이 뉴스가 내 종목에 호재야 악재야?" → 해석 능력 부족
- "나만 이렇게 생각하는 건가?" → 고립된 판단의 불안
- "뭔가 놓치고 있는 거 아냐?" → 정보 FOMO
- "다른 사람들은 어떻게 보고 있지?" → 집단 해석에 대한 갈증

토스증권은 **데이터(시세, 차트, 뉴스)**를 준다. 하지만 **"다른 투자자들의 해석"**은 안 준다.
네이버 금융 댓글은 노이즈 투성이. 종토방은 편향적. 유튜버는 1인 의견.

### 솔루션

> 시장에 무슨 일이 생기면 **가장 먼저 알려주고**, AI 점쟁이 캐릭터들이 **각자 다른 이론/알고리즘으로 예측**하고, 토스 유저들이 **집단 투표**로 해석하고, 그 판단이 **실제로 맞았는지 추적**해주는 곳.

## 3. AI 점쟁이 캐릭터

단순 AI 요약이 아니다. 각 캐릭터가 **논문, 이론, 알고리즘에 기반한 고유 방법론**으로 종목/지수/거시경제를 예측한다. 캐주얼한 점쟁이 느낌이지만 뒤에는 실제 분석 프레임워크가 있다.

| 캐릭터 | 방법론 | 성격 | 예측 스타일 |
|--------|--------|------|-----------|
| 📊 **퀀트봇** | 기술적 분석 (이동평균, RSI, 볼린저밴드, MACD) | 냉철한 숫자충 | "20일 이동평균 돌파. 통계적으로 78% 확률 상승 지속" |
| 🎓 **논문쟁이** | 학술 논문 + 행동경제학 (PEAD, Fama-French, 전망이론) | 아는 척 교수님 | "Bernard & Thomas(1989) PEAD 모델 기준, 실적 서프라이즈 후 60일간 추가 드리프트 예상" |
| 📰 **속보왕** | 뉴스 센티멘트 + 이벤트 드리븐 | 빠른 기자 | "실적 발표 후 1시간 내 긍정 뉴스 비율 83%. 역사적으로 이 비율이면 익일 +1.2%" |
| 🔮 **패턴술사** | 과거 유사 패턴 매칭 + 사이클 이론 | 신비로운 점쟁이 | "2024년 3월과 87% 유사 패턴. 그때는 2주 후 +5.2%. 단, 환율 변수 주의" |
| 🐵 **다트침팬지** | 완전 랜덤 (다트 던지기) | 아무 생각 없음 | "🎯 다트 결과: 호재! (근거: 없음. 근데 나 지난달 적중률 62%였어)" |

### 캐릭터 설계 원칙

- 실제 투자 분석 방법론에 기반. 근거 없는 의견 금지.
- 같은 이슈에 서로 다른 결론이 나와야 함. 전원 호재면 재미없음.
- 예측에 반드시 근거 1줄 이상. "호재" 한 마디 금지.
- 다트침팬지는 베이스라인. 다른 캐릭터가 침팬지보다 못하면 바이럴 콘텐츠.

### 적중률 리더보드

```
이번 달: 📊퀀트봇 72% > 🎓논문쟁이 65% > 📰속보왕 61% > 🐵침팬지 58% > 🔮패턴술사 54%
"침팬지가 패턴술사를 이겼다 ㅋㅋ"
```

## 5. 핵심 기능 (MVP)

### 5.1 오늘의 질문 + 투표
- 매일 오전 9시: 의견 갈릴 만한 이슈 기반 질문 1~2개 (Gemini 생성)
- 구체적 종목/이벤트 기반 ("삼성전자 실적 — 호재?", "코스피 2,800 간다?")
- 투표: 호재 / 악재 / 글쎄 (5초)
- 투표 후 군중 비율 공개 ("67%가 호재"). **투표해야 비율이 보인다.**

### 5.2 AI 점쟁이 예측
- 각 질문에 5캐릭터 서로 다른 예측 + 방법론 기반 근거
- Gemini 캐릭터별 페르소나 프롬프트로 생성

### 5.3 결과 + 적중률 트랙레코드
- 다음날: 실제 시장 vs 군중 vs 캐릭터 비교
- 🎯 적중 / ❌ 빗나감 + 해설
- 캐릭터 리더보드 + 내 적중률

### 5.4 공유 카드 (바이럴)
- 투표 카드: 질문 + 군중 비율 + "너는 어떻게 봐?"
- 결과 카드: 적중/빗나감 + 캐릭터 리포트
- 카카오톡/인스타 공유

## 6. MVP 범위

**IN**: 질문+투표+군중, 5캐릭터 예측, 결과+적중률, 공유 카드, 토스 로그인, AdMob, 면책문구

**OUT (Phase 2+)**: 플래시 투표(속보), 푸시 알림, 관심 종목 개인화, 캐릭터 구독, 한줄 의견, 토스 포인트

## 7. 사용자 여정

```
[첫 진입]
  토스 → 시그널플레이 → 토스 로그인(1탭)
  → 오늘의 질문 ("삼성전자 실적 — 호재?")
  → AI 점쟁이 의견 (퀀트봇: 호재, 패턴술사: 주의, 침팬지: 악재)
  → 투표 (5초) → 군중 67%가 호재
  → 공유 ("너는 어떻게 봐?" → 카톡)

[다음날]
  → 결과 (군중 맞았다 🎯, 침팬지 또 맞혔다 ㅋ)
  → 오늘의 새 질문 → 반복
```

## 9. 바이럴 전략

1. **투표 카드**: "토스 73%가 호재 — 너는?" → 토론
2. **결과 카드**: "군중 적중 🎯 68%" → 놀라움
3. **캐릭터 리포트**: "침팬지가 논문쟁이 이겼다 ㅋ" → 웃김

## 10. 규정 + 리스크

- 캐릭터 예측에 "AI 생성 콘텐츠, 투자 참고용" 면책 필수
- 매수/매도 추천 표현 금지, 암호화폐 직접 언급 금지

| 리스크 | 대응 |
|--------|------|
| 투자 자문 해석 | 면책 + "AI 게임 콘텐츠" 명시 |
