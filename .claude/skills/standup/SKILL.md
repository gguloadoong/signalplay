# Skill: standup — 데일리 스탠드업
**실행 지시:** 매일 작업 시작 전 자동으로 호출됩니다. 각 에이전트는 다음 세 가지를 간결하게 공유합니다:
1. 어제 완료한 작업
2. 오늘 계획된 작업
3. 현재 블로커
*규칙:* 블로커가 발견되면 즉시 해결 방안을 제시하거나 team-meeting 스킬을 호출해야 합니다.
**[자동 체이닝]** 스탠드업이 완료되면 즉시 loop 스킬의 Step 1로 복귀하라.
